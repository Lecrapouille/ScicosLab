log2 - logarithme en base 2

Calling Sequence

y=log2(x)

Parameters

  • x : vecteur ou matrice (réelle ou complexe)

Description

log2(x) calcule le logarithme en base 2 élément par élément : y(i,j)=log2(x(i,j)).

Examples

2.^(log2([1,%i,-1,-%i]))
