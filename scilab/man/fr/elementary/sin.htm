sin - sinus

Calling Sequence

[y]=sin(x)

Parameters

  • x : vecteur ou matrice (réelle ou complexe)

Description

Pour un vecteur ou une matrice, les éléments de sin(x) sont les sinus des éléments de x. Pour le sinus matriciel utiliser la fonction sinm(X).

Examples

asin(sin([1,0,%i]))
