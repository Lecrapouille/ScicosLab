ceil - partie entière supérieure

Calling Sequence

[y]=ceil(x)

Parameters

  • x : matrice réelle
  • y : matrice entière

Description

Renvoie dans y, pour chaque élément de x, le plus petit entier supérieur.

Examples

ceil([1.9 -2.5])-[2,-2]
ceil(-%inf)
x=rand()*10^20;ceil(x)-x
