weakbasis - maxplus matrix image weak basis

Calling Sequence

B=weakbasis(A,[pre])

Parameters

  • A : full maxplus matrix
  • pre : precision
  • B : full maxplus matrix (basis)

Description

B is the weak basis of Im(A) (up to precision pre).

Examples

a=#([2,0;0,1]); b=weakbasis(a)
a=#([2,0,1;0,1,1]); b=weakbasis(a)
a=#([2,0,1,0;0,1,1,%0]); b=weakbasis(a)
