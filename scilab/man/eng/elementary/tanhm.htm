tanhm - matrix hyperbolic tangent

Calling Sequence

t=tanhm(x)

Parameters

  • x,t: real or complex square matrix

Description

tanhm is the matrix hyperbolic tangent of the matrix x.

Examples


A=[1,2;3,4];
tanhm(A)
