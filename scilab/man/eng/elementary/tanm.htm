tanm - matrix tangent

Calling Sequence

[t]=tanm(x)

Parameters

  • x: square real or complex matrix
  • t: square matrix

Description

tanm(x) is the matrix tangent of the square matrix x

Examples


A=[1,2;3,4];
tanm(A)
