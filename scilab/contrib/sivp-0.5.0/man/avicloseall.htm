avicloseall - Close all opened video files/cameras. (experimental)

Calling Sequence

avicloseall()

Description

avicloseall close all opened video files or cameras.

Video support for SIVP is only available when SIVP is compiled with OpenCV which support video I/O.

Examples

	
    n = aviopen(SCI+'/contrib/sivp/images/video.avi');
    im = avireadframe(n); //get a frame
    imshow(im);

    avilistopened()
    aviclose(n);
